Plivo Notifier
==============

Provides [Plivo](https://www.plivo.com) integration for Symfony Notifier.

DSN example
-----------

```
PLIVO_DSN=plivo://AUTH_ID:AUTH_TOKEN@default?from=FROM
```

where:

- `AUTH_ID` is your Plivo Auth ID
- `AUTH_TOKEN` is your Plivo Auth Token
- `FROM` is your sender
